╔══════════════════════════════════════════════════════════════════════════════╗
║                             FLUXO DE TRABALHO                               ║
╚══════════════════════════════════════════════════════════════════════════════╝

1. UPLOAD
   └─ Calcula hash SHA256 do arquivo
      └─ Verifica se é novo
         ├─ Sim: Extrai e valida texto
         └─ Não: Usa cache de sessão

2. EXTRAÇÃO
   ├─ Valida arquivo (vazio, tamanho, formato)
   ├─ Se PDF: BytesIO + PyMuPDF
   ├─ Se TXT: Decodifica (UTF-8 ou Latin-1)
   └─ Valida texto mínimo (10 chars)

3. PREVIEW
   └─ Mostra primeiros 1000 caracteres
      └─ Campo desabilitado (read-only)

4. EDIÇÃO (OPCIONAL)
   └─ Checkbox ativa text area
      └─ Atualiza estado em tempo real

5. ANÁLISE
   ├─ Calcula hash SHA256 do texto
   ├─ Verifica cache (muito rápido se acertou)
   ├─ Se não em cache: Executa análise
   │  ├─ Progresso: 0% → 25% → 50% → 75% → 100%
   │  ├─ Status: ⏳ → 📊 → ✅
   │  └─ Tempo: Cronometrado
   ├─ Salva em cache
   └─ Exibe resultado

6. RESULTADOS
   ├─ Exibe 4 estatísticas
   │  ├─ 🔴 Riscos Altos
   │  ├─ 🟠 Riscos Médios
   │  ├─ 🟢 Riscos Baixos
   │  └─ ⏱️  Tempo
   ├─ Markdown completo
   ├─ Botões: Download | WhatsApp | Cache Info
   └─ Ações: Nova Análise | Fechar

Para testar a aplicação otimizada:

1. Instale as dependências:
   pip install streamlit pymupdf google-generativeai python-dotenv requests

2. Execute a aplicação:
   streamlit run app.py

3. Teste os recursos:
   ✓ Envie um PDF válido → Veja validações
   ✓ Veja preview do texto → Clique na seta para expandir
   ✓ Edite o texto → Ative o checkbox
   ✓ Analyze o contrato → Veja progresso e tempo
   ✓ Veja as estatísticas → 4 métricas em colunas
   ✓ Compartilhe no WhatsApp → Clique no botão
   ✓ Envie o mesmo arquivo novamente → Veja cache (0s)
   ✓ Limpe a análise → Clique no botão 🗑️
